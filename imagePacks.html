How to Create an Image Pack

  1. Open The Image Pack Editor
  2. Give your pack a unique ID, name, and author (you!)
  3. Add locations by clicking the button and then clicking on the map to place a pin.
    (You can change pin's position at anytime by dragging them around the map)
  4. Upload an image and set the difficulty rating for that location
  5. Continue to add locations until you're done!
  6. Download your pack as a ZIP file!
  7. You can test your pack by loading it in the game now!

I made a pack, now what?

Your pack can be merged into the main game!

Here's some ways to submit it:
  • Join the new Discord server and create a post with a link to your pack
  • Create a new issue on GitHub with the title "Image Pack Submission: [Your Pack Name]" and attach your ZIP file
  • Message me on Discord: @logboy2000 with your pack ZIP file
  • Email the ZIP file to me at loganhowarth@icloud.com with the subject "HKGuessr Image Pack Submission"

Submission Requirements:

All community packs that meet these criteria can be added to the main game:
  • No player character visible in the screenshots
  • Roughly 16:9 aspect ratio images (1920x1080 recommended)
  • No UI elements or HUD visible
  • PNG or JPG format
  • Accurate map coordinates for each location
  • Reasonable difficulty ratings
